Detailed Description

GamePlay native activity extension for Android platform.

Handles any platform features that cannot be handled natively in PlatformAndroid.cpp

Developers may choose to directly modify/extend this for any addition platform features that are not offered directly the gameplay3d framework such as platform events, access to android user-interface, life-cycle events for saving game state and custom plug-ins/extensions.
